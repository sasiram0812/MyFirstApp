# 📱 MyFirstApp - React Native (Expo)

🚀 This is my first mobile application built using React Native and Expo.
The project focuses on creating a simple UI using basic components like **View** and **Text**.

* 🧱 Built using React Native core components
* ⚡ Fast development with Expo
* 🎯 Beginner-friendly project

## 🚀 Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Start the App

```bash
npx expo start
```

### 3. Run on Mobile

* Install **Expo Go** on your phone
* Scan the QR code
* View the app instantly 📱
